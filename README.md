# I´M HERE - React Native (rocketseat)
 Projeto 1 da trilha de React Native da plataforma RocketSeat

## Comando utilizado na criação
- npx create-expo-app igniteimhere --tempalte

## Para baixar as dependencias antes de executar o projeto utilzie o seguinte comando:
- npm install 

## Para executar use o seguinte comando:
- npx expo start
